# 워크숍 카지노 게임 운영 가이드

## 개요
워크숍 대기 시간에 참가자들이 모바일로 접속해서 미니게임을 즐기고, 코인 랭킹으로 경쟁하는 킬링타임 콘텐츠입니다.

---

## 사전 준비
1. **Apps Script 배포 확인** — script.google.com에서 최신 Code.gs가 배포되어 있는지 확인
2. **리더보드 초기화** — 어드민 로그인(비번: 1234) → RANKING 탭 → [ RESET ALL ]
3. **사이트 URL 공유** — https://hanbi-kwon.github.io/workshop/ 를 참가자에게 공유 (QR코드 추천)

## 참가자 플로우
1. 사이트 접속 → **🎰 CASINO** 버튼 클릭
2. 닉네임 자동 생성됨 (랜덤 "형용사+동물+숫자", 직접 수정 가능)
3. **코인 100개**로 시작 → 6개 게임 중 자유롭게 선택
4. 코인은 자동으로 서버에 저장됨 (3초마다 동기화)

## 6개 게임 요약

| 게임 | 한줄 설명 | 배당 |
|------|----------|------|
| 🎰 SLOT | 릴 3개 돌려서 맞추기 | 3개 일치 최대 x50, 2개 x2 |
| ✊ DOUBLE UP | 가위바위보, 이기면 더블 | 연승마다 x2 누적, 캐시아웃 가능 |
| 🎡 ROULETTE | 빨강/검정/초록 베팅 | 색 x2, 초록 x9 |
| 🪙 COIN FLIP | 앞뒤 맞추기 더블업 | 연속 x2 누적, 캐시아웃 가능 |
| 🃏 MEMORY | 12장 카드 짝맞추기 | 시도 횟수별 최대 100 |
| 🎲 DICE | 주사위 합 언더/오버 7 | x2, 럭키7 x5 |

## 운영 시나리오

### A. 자유 플레이 (가장 간단)
- "10분 뒤에 코인 1등 발표합니다!" 안내
- 참가자 자유롭게 게임
- 시간 되면 어드민에서 RANKING 탭 → REFRESH → 화면 공유
- 1등에게 소정의 상품

### B. 타임어택
- "5분 안에 코인 가장 많이 모은 사람!"
- 타이머 별도 세팅
- 종료 후 어드민에서 랭킹 확인

### C. 질문 뽑기 사이사이 활용
- 질문 뽑기(GAME START) 사이 대기 시간에 카지노 플레이
- "다음 질문 뽑기 전까지 자유 시간!"

## 어드민 기능
- **RANKING 탭**: 전체 참가자 코인 순위 (REFRESH로 실시간 갱신)
- **RESET ALL**: 리더보드 전체 초기화
- **QUESTIONS 탭**: 질문 관리 (TOTAL/DRAWN/LEFT 통계)

## 주의사항
- 코인이 0이 되면 게임 불가 → 참가자가 CHANGE NAME으로 새 닉네임 만들면 100개로 다시 시작 가능 (막고 싶으면 사전 안내)
- 동시 접속 30명 문제없음
- 모바일/PC 모두 지원
- BGM은 각자 ♪ OFF/ON 토글 가능
